# [Your Name] — Profile (Raw Material)

This file is the comprehensive source of all experience, skills, projects, and background. It is NOT a resume. It is the raw material the skill uses to adapt CVs for specific JDs. Richer and more detailed than a 1-page resume PDF.

## How this file gets created

You do NOT need to fill this manually. The AI can generate it for you:

1. Put your PDF files in this folder (`assets/`). Accepted sources:
   - LinkedIn profile export (PDF). In LinkedIn, go to your profile, click "More", then "Save to PDF".
   - Your current resume or CV (PDF).
   - Any other PDF with your professional info.
2. Tell the AI: "generate my profile from these PDFs" (or "generar mi perfil").
3. The AI will run `pdftotext` on each file, extract your data, and fill this template.
4. The AI will ask you to confirm which skills are truly mastered vs. just learning (proficiency tiers).
5. Review the generated profile. Correct anything that looks wrong.
6. From then on, the skill reads this file automatically when adapting CVs.

You can also edit this file manually at any time to add new experience, projects, or skills.
